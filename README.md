# Future-Hope-iOS


## Install


[CocoaPods - Getting Started](https://guides.cocoapods.org/using/getting-started.html)



>  Clone repo, cd into repo, install

cd Future-Hope
pod init
pod install
open Future-Hope.xcworkspace
